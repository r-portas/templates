# TanStack Start + RSBuild + Mantine Template

A template using TanStack Start and RSBuild, with [Mantine](https://mantine.dev) v9
preconfigured for SSR (PostCSS, `MantineProvider`, and `ColorSchemeScript`).

## Development

```bash
# Install dependencies
bun install

# Start the development server
bun dev

# Build for production
bun run build

# Preview production build
bun preview

# Lint
bun lint

# Format
bun format
```
